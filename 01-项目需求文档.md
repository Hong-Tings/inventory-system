# 进销存管理系统 — 项目需求文档

> 版本：v1.0  
> 最后更新：2026-04-27  
> 状态：初稿

---

## 1. 项目背景

传统中小型企业的进销存管理依赖 Excel 或纸质单据，存在数据不同步、库存不准、追溯困难等痛点。本项目旨在构建一套轻量、可扩展的进销存管理系统，涵盖采购入库、销售出库、库存盘点核心链路，提供 Web 管理后台 + 微信小程序双端操作能力。

---

## 2. 业务目标

| 目标 | 说明 |
|------|------|
| 库存准确率 ≥ 99% | 每次出入库实时更新库存账，盘点误差可追溯 |
| 出入库电子化 | 小程序扫码/手动录入，替代纸质单据 |
| 销售-库存联动 | 出库扣减库存，低于安全库存触发预警 |
| 多仓库支持 | 支持多个仓库与库位管理 |
| 角色权限 | 管理员、库管员、销售员三角色分离 |

---

## 3. 用户角色

| 角色 | 描述 | 操作端 |
|------|------|--------|
| 系统管理员 | 全功能权限：基础数据维护、用户管理、报表查看 | Web |
| 仓库管理员 | 入库、出库、盘点、库存查询 | Web + 小程序 |
| 销售员 | 创建出库单、查看库存 | 小程序 |

---

## 4. 功能需求

### 4.1 基础数据管理（Web 后台）

#### 4.1.1 商品管理
| 功能 | 说明 |
|------|------|
| 商品列表 | 分页展示，支持按名称/编码/分类筛选 |
| 新增商品 | 商品编码（自动生成/手动）、名称、规格、单位、分类、图片、备注 |
| 编辑商品 | 修改商品信息 |
| 启用/停用 | 停用的商品不可用于出入库 |
| 导入/导出 | Excel 批量导入导出商品基础信息 |

#### 4.1.2 分类管理
支持多级商品分类（树形结构），用于商品归类与统计。

#### 4.1.3 仓库管理
| 功能 | 说明 |
|------|------|
| 仓库列表 | 展示所有仓库 |
| 新增仓库 | 名称、编码、联系人、联系电话、地址、状态 |
| 编辑/停用 | 停用后不可进行出入库操作 |

#### 4.1.4 库位管理
| 功能 | 说明 |
|------|------|
| 库位列表 | 按仓库筛选，展示货架/货位 |
| 新增库位 | 所属仓库、库位编码、库位名称（如 A-01-01）、容量、状态 |
| 编辑/删除 | — |

#### 4.1.5 客户/供应商管理
| 功能 | 说明 |
|------|------|
| 客户管理 | 用于销售出库：名称、编码、联系人、电话、地址 |
| 供应商管理 | 用于采购入库：名称、编码、联系人、电话、地址 |

### 4.2 采购入库（小程序为主，Web为辅）

#### 4.2.1 入库流程
1. 选择供应商 → 2. 选择仓库 → 3. 选择库位 → 4. 添加商品（数量、批次号、生产日期） → 5. 确认入库

#### 4.2.2 核心字段
| 字段 | 说明 |
|------|------|
| 入库单号 | 系统自动生成（PR+日期+流水号） |
| 供应商 | 关联供应商基础数据 |
| 仓库 | 入库目标仓库 |
| 库位 | 入库目标库位 |
| 商品明细 | 商品、数量、单价、金额、批次号、生产日期、有效期 |
| 入库日期 | — |
| 操作人 | 当前用户 |
| 备注 | 文本 |

#### 4.2.3 入库单状态
- **草稿** — 保存未提交
- **已入库** — 库存已增加
- **已取消** — 作废

#### 4.2.4 其他
- 入库后可打印入库单（PDF/热敏）
- 支持扫描商品条码快速录入

### 4.3 销售出库（小程序为主，Web为辅）

#### 4.3.1 出库流程
1. 选择客户 → 2. 选择仓库 → 3. 添加商品（选择库存批次） → 4. 确认出库

#### 4.3.2 核心字段
| 字段 | 说明 |
|------|------|
| 出库单号 | 系统自动生成（SO+日期+流水号） |
| 客户 | 关联客户基础数据 |
| 仓库 | 出库仓库 |
| 商品明细 | 商品、数量、单价、金额、批次号 |
| 销售信息 | 销售员、订单号（外部）、销售金额 |
| 出库日期 | — |
| 操作人 | 当前用户 |
| 备注 | 文本 |

#### 4.3.3 出库约束
- 出库数量 ≤ 该仓库该商品可用库存
- 批次优先：支持按先进先出（FIFO）或指定批次出库
- 低于安全库存时弹窗提醒

#### 4.3.4 出库单状态
- **草稿** — 保存未提交
- **已出库** — 库存已扣减
- **已取消** — 作废（回滚库存）

### 4.4 库存管理（Web + 小程序）

#### 4.4.1 库存查询
- 按仓库/商品/库位多维度查询
- 展示：可用库存、在途库存、锁定库存
- 支持导出

#### 4.4.2 库存流水
每一笔出入库均生成库存变动记录，支持按商品/时间范围追溯。

#### 4.4.3 安全库存预警
- 商品可设置安全库存阈值
- 库存低于阈值时系统告警（列表标红/通知）

### 4.5 库存盘点（Web + 小程序）

#### 4.5.1 盘点流程
1. 创建盘点单 → 2. 选择盘点仓库/库位 → 3. 录入实盘数量 → 4. 盘点差异计算 → 5. 审核确认 → 6. 盈亏调整

#### 4.5.2 盘点单字段
| 字段 | 说明 |
|------|------|
| 盘点单号 | 自动生成（CK+日期+流水号） |
| 仓库 | 盘点范围 |
| 盘点方式 | 全盘 / 抽盘 |
| 商品明细 | 账面数量、实盘数量、差异数量、差异原因 |
| 状态 | 盘点中 / 已审核 / 已调整 |
| 盘点人 | — |
| 审核人 | — |

#### 4.5.3 盈亏处理
审核确认后自动生成盈亏调整单，更新库存账。

### 4.6 统计报表（Web）

| 报表 | 说明 |
|------|------|
| 入库报表 | 按日期/供应商/商品汇总入库数量与金额 |
| 出库报表 | 按日期/客户/商品汇总出库数量与金额 |
| 库存周转 | 商品出入库频率分析 |
| 库存预警 | 低于/高于安全库存的商品列表 |
| 操作日志 | 用户操作审计 |

### 4.7 系统管理（Web）

| 功能 | 说明 |
|------|------|
| 用户管理 | 创建/编辑/禁用用户，分配角色 |
| 角色管理 | 自定义角色与权限 |
| 操作日志 | 关键操作审计记录 |
| 系统配置 | 入库单/出库单编码规则、安全库存全局开关等 |

## 5. 非功能需求

| 类别 | 要求 |
|------|------|
| 性能 | 单表 ≤ 100 万条数据时列表查询 < 2s；并发 ≥ 50 |
| 安全 | JWT 登录鉴权；敏感操作记录日志；防 SQL 注入/XSS |
| 可用性 | 核心链路可用 ≥ 99.5% |
| 兼容性 | Web 支持 Chrome/Firefox/Edge 最新版；小程序兼容微信 iOS/Android |
| 响应式 | Web 后台适配桌面端 1920+/平板端 1024+ |

---

## 8. 约束与假设

- 初期不考虑批次拆分/组合；
- 不考虑多单位换算（统一使用基础单位）；
- 不考虑采购订单→入库单的关联（直接创建入库单）；
- 不考虑销售订单→出库单的关联（直接创建出库单）；
- 不支持负库存出库。

## 9. 附录：术语表

| 术语 | 说明 |
|------|------|
| SKU | Stock Keeping Unit，库存量单位 |
| FIFO | First In First Out，先进先出 |
| 在途库存 | 已采购未入库的库存量 |
| 锁定库存 | 已销售未出库的库存量 |
| 可用库存 | 实际可分配库存 = 账面 - 锁定 |
