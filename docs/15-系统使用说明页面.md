# 系统使用说明页面

日期: 2026-05-16

## 新增功能

在 Web 管理端新增「使用说明」独立页面，为所有用户提供完整的系统操作指南。

### 页面位置

- 侧边栏底部新增一级菜单「使用说明」（图标 InfoFilled）
- 路由 `/help`，所有角色（管理员和员工）均可访问

### 页面布局

采用「左侧目录树 + 右侧内容区」两栏布局：

- **左侧侧边栏**（sticky 定位）：顶部搜索框 + 树形导航目录，点击导航项平滑滚动到对应内容区，滚动时自动高亮当前章节
- **右侧内容区**：各章节为可折叠卡片，支持五种内容块渲染

### 内容结构（14 章节）

| 章节 | 内容 |
|------|------|
| 工作台 | 统计卡片、图表、快捷操作、最近动态、库存预警 |
| 商品列表 | 字段释义、操作说明（含条码打印） |
| 商品分类 | 树形结构、增删改查 |
| 供应商管理 | 字段释义、权限说明 |
| 客户管理 | 字段释义、权限说明 |
| 仓库管理 | 多级仓库（1~4 级）、操作说明 |
| 采购入库 | 列表操作、字段、状态流转（含流程图）、审核注意事项 |
| 销售出库 | 同上，含客户/销售员字段、库存扣减说明 |
| 库存列表 | 仓库树展示、批次维度说明 |
| 库存流水 | 字段释义、搜索功能 |
| 库存盘点 | 全盘/抽盘、盘点流程、状态流转 |
| 库存调拨 | 状态流转、注意事项（库存校验） |
| 统计报表 | 入库/出库汇总、周转率、预警说明 |
| 员工管理 | 权限角色说明、操作说明 |
| 操作日志 | 日志字段、筛选说明 |
| 系统配置 | 配置管理说明 |
| 已作废列表 | 支持的单据类型 |
| 权限说明 | 管理员 vs 员工完整权限对照表 |
| 常见问题 | 7 个 FAQ |

### 交互功能

- **搜索**：全文搜索，匹配结果高亮显示，左侧导航自动过滤
- **章节折叠**：点击标题展开/折叠
- **滚动导航**：滚动时左侧目录自动高亮当前章节
- **空状态**：搜索无结果显示空状态提示

## 权限修复

### 取消权限加固

员工可取消草稿单据，但**不可取消已影响库存的单据**（已入库/已出库/已完成状态，涉及库存回滚）。

**前端修改：**
- `PurchaseList.vue` — "取消入库"按钮仅管理员可见
- `SalesList.vue` — "取消出库"按钮仅管理员可见
- `TransferList.vue` — 取消按钮保持对所有角色开放（仅草稿状态）

**后端修改（Service 层）：**
- `PurchaseOrderService.cancel()` — CONFIRMED 状态前加管理员校验
- `SalesOrderService.cancel()` — CONFIRMED 状态前加管理员校验
- `TransferService.cancel()` — CONFIRMED 状态前加管理员校验

### 使用说明文案清理

- 移除了数据导入相关的内容（导入说明、FAQ 导入问题）
- 权限表中取消操作拆分为"取消草稿"和"取消已入库/已出库"
