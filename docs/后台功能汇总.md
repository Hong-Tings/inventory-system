# 进销存管理系统 - 功能汇总

## 一、基础资料管理

| 模块 | API 端点 | 功能 |
|------|----------|------|
| **商品** | `/api/v1/product/page` | 分页查询（支持名称、编码、状态、分类、仓库、价格区间、日期筛选） |
| | `/api/v1/product/list` | 查询所有启用商品 |
| | `/api/v1/product/{id}` | 获取单个商品详情 |
| | `/api/v1/product` | 新增/更新商品 |
| | `/api/v1/product/{id}` | 删除（作废）商品 |
| | `/api/v1/product/barcode/download` | 批量生成商品条码（ZIP） |
| | `/api/v1/product/barcode/{id}` | 单个商品条码 |
| **商品分类** | `/api/v1/category/tree` | 获取分类树形结构 |
| | `/api/v1/category` | 新增/更新/删除分类 |
| **供应商** | `/api/v1/supplier/page` | 分页查询（支持名称、联系人、电话、地址筛选） |
| | `/api/v1/supplier/list` | 查询所有启用供应商 |
| | `/api/v1/supplier` | 新增/更新/删除供应商 |
| **客户** | `/api/v1/customer/page` | 分页查询（支持名称、联系人、电话、地址筛选） |
| | `/api/v1/customer/list` | 查询所有启用客户 |
| | `/api/v1/customer` | 新增/更新/删除客户 |
| **仓库** | `/api/v1/warehouse/page` | 分页查询（支持名称、联系人、电话、地址、状态筛选） |
| | `/api/v1/warehouse/list` | 查询所有启用仓库 |
| | `/api/v1/warehouse` | 新增/更新/删除仓库 |
| **自动编码** | 后端自动生成 | 商品:GD, 仓库:WH, 供应商:SP, 客户:CT + yyyyMMdd + 3位流水 |

## 二、业务单据

| 模块 | API 端点 | 功能 |
|------|----------|------|
| **采购入库** | `/api/v1/purchase-order/page` | 分页查询（支持单号、供应商、仓库、数量、金额、日期筛选） |
| | `/api/v1/purchase-order/{id}` | 获取详情（含商品明细） |
| | `/api/v1/purchase-order` | 新增（含明细） |
| | `/api/v1/purchase-order/{id}/draft` | 更新草稿 |
| | `/api/v1/purchase-order/{id}/submit` | 提交入库（触发库存变动） |
| | `/api/v1/purchase-order/{id}/cancel` | 取消 |
| | `/api/v1/purchase-order/{id}/void` | 作废（管理员） |
| | `/api/v1/purchase-order/batch-void` | 批量作废 |
| | 状态: 0=草稿, 1=已入库, 2=已取消, 3=已作废 |
| **销售出库** | `/api/v1/sales-order/page` | 分页查询（支持单号、客户、仓库、数量、金额、销售员、日期筛选） |
| | `/api/v1/sales-order/{id}` | 获取详情（含商品明细） |
| | `/api/v1/sales-order` | 新增（含明细，FIFO出库） |
| | `/api/v1/sales-order/{id}/draft` | 更新草稿 |
| | `/api/v1/sales-order/{id}/submit` | 提交出库（FIFO扣减库存） |
| | `/api/v1/sales-order/{id}/cancel` | 取消 |
| | `/api/v1/sales-order/{id}/void` | 作废（管理员） |
| | `/api/v1/sales-order/batch-void` | 批量作废 |
| | 状态: 0=草稿, 1=已出库, 2=已取消, 3=已作废 |

## 三、库存管理

| 模块 | API 端点 | 功能 |
|------|----------|------|
| **库存查询** | `/api/v1/inventory/page` | 分页查询（支持商品名称、仓库筛选、汇总数量） |
| **库存流水** | `/api/v1/inventory/log/page` | 分页查询（支持商品名、仓库、类型、单号、操作人、日期筛选） |
| | 类型中文：采购入库、销售出库、调拨入库、调拨出库、取消入库、取消出库、盘点调整 |
| **库存盘点** | `/api/v1/stock-take/page` | 分页查询（支持单号、仓库、方式、数量、差异、状态筛选） |
| | `/api/v1/stock-take/{id}` | 获取详情（含盘点明细） |
| | `/api/v1/stock-take` | 新增盘点单 |
| | `/api/v1/stock-take/item` | 添加盘点明细 |
| | `/api/v1/stock-take/item/{itemId}` | 删除/更新盘点明细 |
| | `/api/v1/stock-take/{id}/approve` | 审核盘点单 |
| | `/api/v1/stock-take/{id}/adjust` | 盘点调整（更新库存） |
| | `/api/v1/stock-take/{id}/void` | 作废（管理员） |
| | `/api/v1/stock-take/batch-void` | 批量作废 |
| | 状态: 0=盘点中, 1=已审核, 2=已调整, 3=已作废 |
| **库存调拨** | `/api/v1/transfer/page` | 分页查询（支持单号、调出仓库、调入仓库、数量、状态、日期筛选） |
| | `/api/v1/transfer/{id}` | 获取详情（含调拨明细） |
| | `/api/v1/transfer` | 新增调拨单 |
| | `/api/v1/transfer/{id}/draft` | 更新草稿 |
| | `/api/v1/transfer/{id}/submit` | 提交调拨（触发库存变动） |
| | `/api/v1/transfer/{id}/cancel` | 取消 |
| | `/api/v1/transfer/{id}/void` | 作废（管理员） |
| | `/api/v1/transfer/batch-void` | 批量作废 |
| | 状态: 0=草稿, 1=已完成, 2=已取消, 3=已作废 |

## 四、统计报表

| 模块 | API 端点 | 功能 |
|------|----------|------|
| **工作台** | `/api/v1/dashboard/stats` | 商品总数、仓库数、今日入库单数/数量、今日出库单数/数量、预警数 |
| **入库汇总** | `/api/v1/report/purchase-summary` | 近N天入库趋势（按日期统计单数、数量、金额） |
| **出库汇总** | `/api/v1/report/sales-summary` | 近N天出库趋势 |
| **库存预警** | `/api/v1/report/inventory-alert` | 低于安全库存的商品列表 |
| **周转率** | `/api/v1/report/turnover-rate` | 商品周转率排行 |

## 五、系统管理

| 模块 | API 端点 | 功能 |
|------|----------|------|
| **员工管理** | `/api/v1/user/page` | 分页查询用户 |
| | `/api/v1/user` | 新增/更新/删除用户（管理员） |
| **操作日志** | `/api/v1/system/log/page` | 查询操作日志（支持模块、日期筛选） |
| **系统配置** | `/api/v1/system/config` | 获取/更新系统配置 |
| **已作废列表** | `/api/v1/recycle/list` | 获取所有已作废数据（分类返回） |

## 六、认证与通用

| 模块 | API 端点 | 功能 |
|------|----------|------|
| **登录** | `/api/v1/auth/login` | 用户名密码登录（BCrypt验证） |
| | `/api/v1/auth/userinfo` | 获取当前用户信息 |
| | `/api/v1/auth/logout` | 退出 |
| **文件上传** | `/api/v1/file/upload` | 上传图片（存本地，返回URL） |
| **Excel导出** | 各模块 `/export` | 导出中文表头的 Excel |

## 七、小程序适用功能

整理适合在手机上操作的功能（标记优先级）：

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P0 | 采购入库 新建/列表/详情 | 到货时手机录入 |
| P0 | 销售出库 新建/列表/详情 | 开单时手机录入 |
| P0 | 库存查询 | 随时查库存 |
| P0 | 库存盘点 新建/录入 | 仓库现场盘点 |
| P0 | 扫码识别商品 | 扫码快速定位商品 |
| P1 | 库存调拨 新建/列表/详情 | 仓库间调货 |
| P1 | 库存流水 | 查看库存变动记录 |
| P2 | 工作台首页 | 查看今日概览 |
| P2 | 商品列表 | 查看/搜索商品信息 |
| P2 | 库存预警 | 查看预警商品 |
