# 进销存管理系统 — 项目成果总结

> 日期：2026-04-29 | 版本：v2.0

---

## 一、系统概述

一套完整的企业进销存管理系统，覆盖**采购入库 → 销售出库 → 库存盘点 → 库存调拨 → 统计报表**核心业务全链路。支持 PC 管理后台 + 微信小程序双端操作。

## 三、已实现功能清单

### 3.1 基础资料管理
- **商品管理** — 分页查询、新增/编辑/作废、批量启用/停用、图片上传、条码生成（单个+批量ZIP下载）、Excel导出
- **商品分类** — 树形结构、CRUD、父子级关联、作废保护（有商品引用的分类禁止作废）
- **供应商管理** — 分页查询、CRUD、批量作废、Excel导出、时间升降序
- **客户管理** — 分页查询、CRUD、批量作废、Excel导出、时间升降序
- **仓库管理** — 分页查询、CRUD、批量作废、列表汇总（总数+金额）、时间记录
- **自动编码** — 后端自动生成（GD/WH/SP/CT + yyyyMMdd + 3位流水），禁止手动输入，保证唯一

### 3.2 业务单据
- **采购入库** — 新建（含明细）/草稿编辑/提交入库（库存增加）/取消（回滚库存）/作废/批量作废、状态流转（草稿→已入库→已取消→已作废）、列表汇总
- **销售出库** — 新建（含明细）/草稿编辑/提交出库（FIFO扣减库存）/取消（回滚库存）/作废/批量作废、安全库存预警、列表汇总

### 3.3 库存管理
- **库存查询** — 多维度查询（商品/仓库）、库存预警变色显示、汇总（可用/锁定/可发）
- **库存流水** — 按商品/仓库/类型/单号/操作人/时间筛选、全类型追溯（采购入库/销售出库/调拨出入库/取消/盘点调整）
- **库存盘点** — 创建盘点单（全盘/抽盘）、明细录入、审核、调整（更新库存）、作废
- **库存调拨** — 创建调拨单、提交（源仓库扣减+目标仓库增加）、取消（双向回滚）、禁止同仓库调拨

### 3.4 统计报表
- **工作台仪表盘** — 商品总数、仓库数、今日入库单数/数量、今日出库单数/数量、预警数
- **入库汇总** — 近N天入库趋势（按日期统计单数、数量、金额）
- **出库汇总** — 近N天出库趋势
- **库存预警** — 低于安全库存的商品列表
- **周转率** — 商品周转率排行

### 3.5 系统管理
- **用户管理** — CRUD、账号状态管理
- **角色管理** — 角色定义与分配
- **操作日志** — AOP切面自动记录写操作、按模块/日期筛选
- **系统配置** — 编码规则、业务参数配置
- **已作废列表** — 所有作废数据分类展示（含作废时间、作废原因），支持恢复

### 3.6 小程序端
- 采购入库新建/列表/详情
- 销售出库新建/列表/详情
- 库存查询/流水/预警
- 库存盘点录入
- 库存调拨

## 四、技术实现

### 后端
- **框架**：Spring Boot 3.2.5 + JDK 17 + MyBatis-Plus 3.5.5
- **认证**：Sa-Token 1.37 + JWT（无状态，无需Redis）
- **接口文档**：Knife4j (Swagger)
- **工具**：Hutool 5.8、EasyExcel 3.3、ZXing（条码生成）
- **数据库**：MySQL 8.0，21张表，含完整索引设计

### 前端（PC后台）
- **框架**：Vue 3.4 + TypeScript + Element Plus + Pinia
- **路由**：20+路由，含登录守卫、角色权限控制
- **API**：Axios封装，含自动Token注入、参数清理、401拦截

### 前端（小程序）
- **框架**：UniApp 3.x + Vue 3 + Pinia
- **页面**：19个页面，5个Tab导航（首页/入库/出库/库存/我的）

## 五、关键业务流程验证

```
登录 → 商品管理 → 采购入库 → 库存增加 → 
销售出库 → 库存扣减（FIFO）→ 安全库存预警 →
库存盘点 → 盈亏调整 → 库存流水追溯 →
库存调拨 → 调出扣减 → 调入增加
```

全链路已实现并验证通过。
