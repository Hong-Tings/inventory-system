## 六、验收标准

| 验收项 | 标准 |
|--------|------|
| Docker 部署 | `docker-compose up -d` 一键启动，所有服务正常运行 |
| 核心链路 | 登录→商品管理→入库→出库→盘点 全流程无报错 |
| 并发安全 | 同一商品同时出库不会出现负库存 |
| 图片上传 | 商品可上传图片，列表和详情正常显示 |
| Excel | 商品可导出为 xlsx，可导入 xlsx 批量创建商品 |
| FIFO 出库 | 先入库的批次优先出库，也可手动指定批次 |
| 部署文档 | 按文档步骤操作，一个新环境可在 30 分钟内完成部署 |
